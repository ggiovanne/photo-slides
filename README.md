# Foto Slides

Aplicativo para exibir slideshow de fotos enviadas por usuários em eventos, com QR code para upload rápido.

## Funcionalidades

- Crie um nome para o seu evento
- Inicie o slideshow em tela cheia
- QR code fixo para os convidados acessarem a página de upload
- Upload de fotos (selecionar arquivo ou usar câmera)
- Atualização automática do slideshow com novas fotos

## Como Hospedar Gratuitamente no Vercel

1. Crie uma conta no [Vercel](https://vercel.com/) (gratuita)
2. Crie uma conta no [Vercel Storage](https://vercel.com/docs/storage/vercel-blob) e ative o **Vercel Blob** (gratuita para até 5GB)
3. Faça upload deste projeto para um repositório GitHub/GitLab/Bitbucket
4. Importe o repositório no Vercel
5. No painel do Vercel, conecte o Vercel Blob ao seu projeto
6. Deploy automático!

## Tecnologias

- Next.js 14
- TypeScript
- Tailwind CSS
- Vercel Blob (armazenamento de arquivos)
- Vercel (hospedagem)
